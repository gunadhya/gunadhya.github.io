    Android Home launcher from scratch

    Jan 20, 2019
    Development, Android Development
    android, android-development, launcher, kotlin, development, MVVM

    Intro #

    An attempt to create a simple android home screen launcher in kotlin.

    About #

    Features #

    • Grid apps
    • List apps
    • Abilty to search for apps
    • Gestures Support (Swipe Left to clear text, Swipe down to disable keyboard)
    • App Size (small, medium, large)
    • Always keyboard to search quickly
    • Long press an app to uninstall the app or view system app settings
    • Add blur to background (WIP)

    Built using #

    • RecyclerView
    • MVVM architecture
    • Kotlin
    • Android Studio
    • Android ViewModel
    • PackageManager
    • Android Navigation
    • Android Jetpack
